### README

#### Brief
My latest attempt at making my dream scripting language.

#### Requirements
 - Clang 21+
 - Ninja (latest is best)
 - CMake 4.2
 - Bash / ZSH

#### Getting Started
 1. Enable the `utility.sh` shell script.
 2. Run `./utility.sh build release Ninja`.
 3. To run Edna code: `./build/ednam -r <script>`
